Como funciona o controle de acesso

Cada usuário faz login com seu login e senha. O sistema verifica as credenciais e exibe apenas os setores cadastrados para ele aqui. Se o usuário não estiver cadastrado, o acesso é bloqueado automaticamente. Setores vazios = acesso a todos os setores permitidos.
